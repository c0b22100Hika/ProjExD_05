# ゲームのタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
## ゲームの概要
こうかとんがゾンビに襲われるので、倒すゲーム
## ゲームの実装
### 共通基本機能
* キャラクターを管理するクラス
* プレイヤーに関するクラス
* ゾンビに関するクラス
* エフェクトに関するクラス
* ビームに関するクラス
* スコアに関するクラス
### 担当追加機能
* BOSSゾンビに関するクラス
* クリア条件
* 可変フレームレート
